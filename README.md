# Todoit - Todo List App

Todoit is a todo list app made as per the curriculum of [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-todo-list). The app allows the users to create, update, and delete todo list tasks. Mark a todo as completed or in progres. You can also create different projects and assign new todos to the projects. The app also has build-in crash handling incase the user delete all the data in the local storage.

## Features

- **Local Storage**: The app uses local storage to store and render todos and projects.
- **ES6 Modules**: The code uses the latest ES6 modules to organize the files.
- **Webpack**: Bundles JavaScript, CSS, and image files for optimized performance.
- **Dynamic Display**: All page contents are generated dynamically via JavaScript.
- **OOP**: The code is structured following the principles of object oriented programming.
